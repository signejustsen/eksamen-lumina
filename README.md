Oprydning i selektorer og fjernelse af ubrugt kode.

Ensartet brug af CSS-variabler til farver, typografi, størrelser og interaktionsstilarter.

Variablerne i :root bruges nu konsekvent (fx --pink-color, --font-size-large, --font-weight-bold).

Dette har gjort hele stylingen mere konsistent og markant lettere at vedligeholde.

Rettelser af layoutfejl (fx social proof og footer, som tidligere flyttede sig utilsigtet).
